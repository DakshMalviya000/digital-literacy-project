This project is based on Digital Literacy and aims to develop essential skills required to use digital technologies effectively and responsibly. As a Student Digital Ambassador, I completed various tasks related to digital awareness, online platforms, communication, and cybersecurity.

The project helped me understand the importance of safe internet practices, professional online presence, and responsible digital communication. Through this project, I also learned how to use different online tools and platforms that are useful for academic and professional growth.

Task 1 :-
In Task 1, I created a Digital Literacy infographic using an online design tool. The infographic explains the concept of digital literacy, safe internet practices, and email etiquette.

While creating the infographic, I focused on making the content simple, clear, and visually appealing. I included important points like using strong passwords, avoiding phishing, and maintaining professional communication.

This task helped me understand how to present information in a visual format and improved my creativity. I also learned how to use design tools effectively.

Task 2 :-
In Task 2, I created my digital portfolio using platforms like GitHub and LinkedIn. GitHub is used to manage and showcase projects, while LinkedIn helps in professional networking.

I set up my profiles and added basic information such as education and skills. This task helped me understand the importance of having an online presence.

In the future, I plan to use these platforms to showcase my projects, connect with professionals, and explore career opportunities.

Task 3 :-
In Task 3, I explored coding and collaboration platforms. I used HackerRank to solve a beginner-level problem, which helped me understand basic programming concepts.

I also created a Google Form quiz on Digital Literacy. This helped me learn how to create surveys and collect responses using Google Sheets.

This task improved my technical skills and gave me practical experience with useful online tools.

Task 4 :-
In Task 4, I learned about professional communication and email etiquette. I wrote two formal emails and created a social media etiquette checklist.

I understood the importance of using a proper subject line, polite language, and clear communication. This task helped me improve my communication skills and taught me how to behave responsibly online.

Task 5 :-

I was surprised to learn how easily people can fall into online scams. This task made me more aware of online threats and encouraged me to follow safe practices while using digital platforms.

CONCLUSION :-
Overall, this project helped me gain knowledge about digital literacy and its importance in everyday life. I learned about safe internet practices, professional communication, and online platforms.

This project also improved my technical and communication skills. I will apply these learnings in my academic and professional journey.
